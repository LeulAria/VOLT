# Security Policy

## Reporting a Vulnerability
Please **do not** open a public GitHub issue for security vulnerabilities.

Email the maintainers directly with:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and aim to patch critical issues within 7 days.

## Scope
- Electron IPC boundary security
- PTY command injection
- Browser webview sandbox escapes
- Local file system access controls
